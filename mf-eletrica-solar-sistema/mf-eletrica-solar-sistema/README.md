# Sistema MF Elétrica Solar

Sistema web em React + Vite para gestão de alertas, agenda técnica, relatórios diários, clientes, painéis de monitoramento e checklist.

## Como rodar no computador
1. Instale o Node.js: https://nodejs.org
2. Abra a pasta do projeto no VS Code.
3. No terminal, rode:
   npm install
   npm run dev

## Como publicar na Vercel
1. Acesse https://vercel.com e crie uma conta.
2. Crie um repositório no GitHub e envie estes arquivos.
3. Na Vercel, clique em Add New Project.
4. Importe o repositório.
5. Clique em Deploy.

Depois disso, a Vercel vai gerar um link online para acessar pelo celular ou computador.
